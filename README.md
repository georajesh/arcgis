# 🗺️ ArcGIS Projects & Automation

Welcome to the **ArcGIS** repository — a curated collection of scripts, notebooks, and workflow samples focused on automating and enhancing GIS tasks using **ArcGIS Pro**, **Python (ArcPy)**, and spatial data tools.

This repo includes:

- 🔁 **MXD to APRX Migration Workflows**  
  Scripts for bulk conversion of legacy `.mxd` files to `.aprx`, including read-only map handling and data source repair.

- 🧹 **Data Cleanup & Repair Tools**  
  Code to detect and fix broken layer connections, automate reconnections, and streamline common ArcGIS maintenance tasks.

- 🛠️ **ArcGIS Pro Automation Examples**  
  Jupyter notebooks and code snippets to manage maps, layers, layouts, and metadata with `arcpy`.
